Данный код может проверить промежуток памяти на доступность, путем записи и сравнения содержимого. Если данные 
различаются будет выведен регион памяти, который доустпен для записи в формате Region: 0x.. - 0x.
Настраивается все через переменные в коде(scan_start, scan_end, step), которые отвечают за адресс с которого
начинается проверка, заканчивается и шаг с которым будет идти по памяти. Посел этого нужно использовать makefile, для
компиляции и запуска.
